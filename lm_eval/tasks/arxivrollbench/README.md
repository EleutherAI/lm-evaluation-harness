# ArxivRollBench

ArxivRollBench is a dynamic private benchmark for measuring overestimation in large language model evaluation. It is generated by ArxivRoll, a one-time-pad-inspired framework that constructs fresh benchmarks from recent arXiv papers so that evaluated models are less likely to have seen the exact test cases during training or benchmark-aware tuning.

The core idea is to periodically roll the benchmark forward with newly available papers, transform paper text into private evaluation items, and compare model behavior on these fresh items against public benchmark performance. This makes ArxivRollBench useful for studying benchmark contamination, benchmark overfitting, and inflated public-leaderboard estimates.

ArxivRollBench uses three SCP construction strategies:

- sequencing, which asks models to recover the correct order of shuffled text fragments;
- cloze, which asks models to select the correct ordering around masked or removed sentences;
- prediction, which asks models to choose the most plausible next sentence from candidate continuations.

This integration includes ArxivRollBench 2024b, 2025a, and 2026a. Each release covers eight arXiv domains (`cs`, `q-fin`, `math`, `physics`, `stat`, `q-bio`, `econ`, and `eess`) and three SCP task types:

- `s`: sequencing, where a model selects the correct order for shuffled text fragments.
- `c`: cloze, where a model selects the correct ordering of masked sentences.
- `p`: prediction, where a model selects the most plausible next sentence.

## Splits and task names

The default task names use the compact `-50` evaluation split for cost-controlled evaluation:

```bash
lm_eval --model hf --model_args pretrained=<model> --tasks arxivrollbench2026a
lm_eval --model hf --model_args pretrained=<model> --tasks arxivrollbench2026a_cs_s
```

Full-split tasks are available with the `_full` suffix:

```bash
lm_eval --model hf --model_args pretrained=<model> --tasks arxivrollbench2026a_full
lm_eval --model hf --model_args pretrained=<model> --tasks arxivrollbench2026a_cs_s_full
```

Available aggregate groups:

- `arxivrollbench`: all included releases on the compact `-50` split.
- `arxivrollbench_full`: all included releases on the full split.
- `arxivrollbench2024b`, `arxivrollbench2025a`, `arxivrollbench2026a`: compact release-level groups.
- `arxivrollbench2024b_full`, `arxivrollbench2025a_full`, `arxivrollbench2026a_full`: full release-level groups.
- `arxivrollbench{release}_{s,c,p}` and `arxivrollbench{release}_{s,c,p}_full`: SCP-type groups.
- `arxivrollbench{release}_{cs,q_fin,math,physics,stat,q_bio,econ,eess}` and matching `_full` variants: domain groups.

## Metric

Tasks use generation plus exact match after regex extraction. Sequencing and cloze tasks extract `Selection 1` through `Selection 4`; prediction tasks extract `A`, `B`, `C`, or `D`.
