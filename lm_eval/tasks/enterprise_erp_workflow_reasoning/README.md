# Enterprise ERP Workflow Reasoning

**Author:** Venkata Ramachandra Karthik Chundi (venkatakarthikchundi@gmail.com)

## Task Description

Tests LLM reasoning on enterprise ERP business process workflows, approval hierarchies, document relationships, and process sequencing. Scenarios are drawn from real-world Oracle ERP Cloud deployment contexts.

50 multiple-choice questions with 4 options each, covering:

- **Procurement:** procure-to-pay, three-way match, BPA, approved supplier lists, invoice holds
- **HCM:** work relationships, assignments, element entries, terminations, retroactive payroll
- **General Ledger:** journal posting, period close, reversals, accruals, allocations, revaluation
- **Fixed Assets:** mass additions, depreciation, retirements, CIP capitalization
- **SCM:** transfer orders, receiving, cycle counts, lot control, RMA, ship confirmation
- **EPM Planning:** substitution variables, budget promotion, spreading, approval units
- **OIC:** connections, integration activation
- **CX Sales:** leads, quotes, deal registrations
- **Project Management:** tasks, bill rate schedules
- **Manufacturing:** work orders, MRP
