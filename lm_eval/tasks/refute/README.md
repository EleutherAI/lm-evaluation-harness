# REFUTE tasks

Judge-free tasks from the [REFUTE benchmark](https://huggingface.co/datasets/BGPT-OFFICIAL/refute) for scientific critique and epistemic calibration.

| Task | Description |
|------|-------------|
| `refute_soundness` | Binary sound/flawed classification over study summaries (`acc`) |
| `refute` | Group containing `refute_soundness` |

Dataset: `BGPT-OFFICIAL/refute` config `refute_soundness`.

Technical report: https://huggingface.co/datasets/BGPT-OFFICIAL/refute/blob/main/TECHNICAL_REPORT.md
